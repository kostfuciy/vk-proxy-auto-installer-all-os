# Auto-Installer WireGuard + VK TURN Proxy

This script allows you to deploy your own VPN server with WireGuard and the vk-turn-proxy utility from [cacggghp](https://github.com/cacggghp/vk-turn-proxy) or a fork from [kiper292](https://github.com/kiper292/vk-turn-proxy) in one command to bypass blocking and whitelisting. For more information, visit the authors' repositories.

The script fully automates the routine: it installs packets, forwards ports, and creates a convenient console control panel.

## 🛠 Installation

Run the command below with root (clean Ubuntu/Debian OS recommended):

```bash
bash <(curl -sL https://raw.githubusercontent.com/kostfuciy/vk-proxy-auto-installer-all-os/refs/heads/main/vk-proxy-auto-installer-all-os.sh)
```
