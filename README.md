# Автоустановщик VK TURN Proxy

Это модифицрованная версия скрипта от [NedgNDG](https://github.com/NedgNDG/vk-proxy-auto-installer), которая позволяет поставить его на любой дистрибутив Linux.


## 🛠 Установка

Запустите команду ниже на вашем сервере:

```bash
bash <(curl -sL https://raw.githubusercontent.com/kostfuciy/vk-proxy-auto-installer-all-os/refs/heads/main/vk-proxy-auto-installer-all-os.sh)
```
